# 开发进度

## 项目目标

本项目目标是开发一个基于 **ESP32-S3-CAM + Linux 服务端 + YOLOv8** 的边缘视觉安全告警系统。

系统最终链路：

```text
ESP32-S3-CAM 拍照
-> Wi-Fi 上传图片
-> Linux 服务端接收图片
-> YOLOv8 目标检测
-> 告警规则判断
-> 数据库存储
-> Web 页面展示
-> 告警处理或联动
```
